What this app proves

Demo-machine can target resiliently using labels, roles, and test IDs. This page includes: a checkbox, a select, a file input, and HTML5 drag and drop.
Tip: Use `target.by: role` or `target.by: label` first. Keep CSS selectors as a last resort.
